# Rhuan Silva 💻

**`Software Engineer`**

Atualmente, dedico parte do meu tempo aos estudos de programação, desenvolvimento de software e novas tecnologias, sempre buscando aprender, evoluir e transformar ideias em projetos que unam tecnologia, criatividade e propósito.
